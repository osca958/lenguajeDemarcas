¿Qué es CSS?

CSS es el lenguaje utilizado para aplicar estilos a las páginas web. Permite definir colores, tamaños, márgenes, posicionamiento y muchos más

¿Por qué usar CSS?

Separar el contenido (HTML) del estilo (CSS) dacilita el mantenimiento, mejora la accesibilidad y permite el diseño responsivo

Modelo de caja

Todo elemento HTML se comporta como una caja. Con CSS podemos definir sus ancho, alto, margen, borde y relleno

Flexbox

Un sistema de diseño flexible que permite distribuir y alinear elementos fácilmente dentro de un contenedor

Grid

Grid es una poderosa herramienta para contruir diseños complejos con filas y columnas
